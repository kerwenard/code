# 字符串格式

* f
    格式化字符串
```python
t='abc'
print(f'{t}') # python 3.6以上
# ----------------------------------------------------------------
abc
```

* u
    中文字符串，Unicode编码
```python
print(u'中文')
# ----------------------------------------------------------------
中文
```

* r
    申明普通字符串，即无效化转移字符
```python
print(r'\r\n')
# ----------------------------------------------------------------
\r\n
```
